== Kletterverein Management
Das Klettervereinsmanagement ist dazu da, damit die Kletterer sehen können welche Routen frei sind und welche besetzt
sind. Die Kletterer müssen dazu immer die Route eintragen, die sie dann klettern möchten und sie dann wieder austragen.
Zudem können die Klettervereinsbesitzer sehen welche Route gerne und oft geklettert wird und welche nicht. (Noch zu implementieren)
